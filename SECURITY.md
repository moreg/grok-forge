# Security Policy

If you believe you have found a security vulnerability in **Grok Forge**,
please report it privately.

- Prefer contacting the repository owner via GitHub Security Advisories on
  [moreg/grok-forge](https://github.com/moreg/grok-forge/security), or
- Open a **private** report to the maintainer — do not file a public issue
  with exploit details.

We will acknowledge reports as capacity allows.
